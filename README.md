
This project is a movie rental application developed using React. It features a user interface with components for listing movies, filtering by genre, searching, and pagination. The application also includes functionalities for sorting movies, deleting entries, and managing user authentication with a login form that uses Joi for validation. The project demonstrates a comprehensive CRUD operation on movie data and integrates Bootstrap for styling and responsive design.
